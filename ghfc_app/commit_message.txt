feat(i18n,ui,avatar,provider): 本地化、过滤控件、头像重试与分页优化

  - 通过 ValueNotifier 驱动语言切换与 UI 重建
  - 在 CustomAppBar、CombinedFilterWidget、EventProvider 中集成

- UI 组件
  - 顶部栏新增渐变标题、语言切换弹窗与仓库入口

  - UserCard 事件分组与弹出链接，提升可读性和可操作性

- 架构与实现
  - 采用 Provider + ChangeNotifier 管理事件与过滤状态
  - 统一事件短名、样式、文档链接，降低 UI 层复杂度
  - 通过分页与懒加载实现无限滚动

- 性能优化
  - 分页渲染与按需构建弹出项减少初始与滚动开销
  - 折叠顶部栏与紧凑 legend 提升移动端可视与响应

  - URL 跳转前 canLaunchUrl 校验
  - 头像失败有限重试与手动刷新，避免资源滥用

  - 配置 flutter_launcher_icons 使用自定义图标与背景色
